# Versions

* latest (based on debian:stable)

# Image

Install sqlite3 command line.  Use make install to setup the command.

```bash
docker pull nouchka/sqlite3
```

# Build

```bash
make build
```

# Use


```bash
docker-compose run sqlite3
```
